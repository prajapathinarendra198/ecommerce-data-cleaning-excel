# Task 1 - Data Immersion & Wrangling

This project was completed as part of my Data Analyst Internship at ApexPlanet Software Pvt. Ltd.

## Objective
To clean, validate, and transform an E-commerce Orders dataset using Microsoft Excel.

## Tasks Performed
- Checked missing values
- Validated Total_Price using formulas
- Verified duplicate records
- Cleaned and standardized Order_Date
- Extracted Year, Month, and Day from dates
- Created a Data Dictionary

## Tools Used
- Microsoft Excel
- Conditional Formatting
- Excel Formulas

## Files Included
- Cleaned Dataset (.xlsx)
- Walkthrough Video
